# Kashlog

A new Flutter project for expense tracking and financial management.

## Requisitos de Instalación

Para ejecutar este proyecto localmente, asegúrate de tener instalado lo siguiente:

1.  **Flutter SDK**: Sigue la guía oficial de instalación en [flutter.dev](https://docs.flutter.dev/get-started/install).
2.  **Dart SDK**: Incluido con el SDK de Flutter.
3.  **Editor de Código**: Se recomienda VS Code o Android Studio con los plugins de Flutter y Dart.
4.  **Configuración de Dispositivo**: Un emulador de Android, un simulador de iOS o un dispositivo físico conectado.

## Pasos para la Instalación

1.  Descarga o clona este repositorio.
2.  Abre una terminal en la carpeta raíz del proyecto.
3.  Ejecuta el siguiente comando para instalar todas las dependencias:
    ```bash
    flutter pub get
    ```
4.  Para ejecutar la aplicación, usa:
    ```bash
    flutter run
    ```

## Dependencias del Proyecto

Este proyecto utiliza las siguientes librerías:

- **cupertino_icons**: Iconos estilo iOS.
- **sqflite**: Base de datos local SQLite.
- **path**: Manipulación de rutas de archivos.
- **provider**: Gestión de estado.
- **fl_chart**: Gráficos dinámicos y hermosos.
- **intl**: Internacionalización y formato de fechas/números.
- **sqflite_common_ffi_web**: Soporte de SQLite para web.
- **shared_preferences**: Almacenamiento local persistente de datos simples.
- **countup**: Animaciones de conteo de números.
- **flutter_animate**: Animaciones fluidas para la UI.
- **csv**: Parsing y generación de archivos CSV.
- **share_plus**: Funcionalidad para compartir archivos y texto.
- **flutter_local_notifications**: Notificaciones locales en el dispositivo.
- **timezone**: Manejo de zonas horarias para las notificaciones.
- **google_ml_kit**: Reconocimiento de texto (OCR) y otras herramientas de ML.
- **image_picker**: Selección de imágenes desde la galería o cámara.
- **home_widget**: Integración con widgets de la pantalla de inicio.

## Dependencias de Desarrollo

- **flutter_test**: Framework para pruebas unitarias y de widgets.
- **flutter_lints**: Reglas de estilo y mejores prácticas de código.
- **flutter_launcher_icons**: Generación automática de iconos de la aplicación.
